# Longest Word

Given an list of strings, find the longest word!

---

## 1. Details

The list of words will be given to you as a `list` called `array`. Create a program that finds the longest word in the list.

---

## 2. Test Cases

```
wordList = ["Nimble", "Yellow", "Curious"]
```

```
Curious
```
